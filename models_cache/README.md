# models_cache

This folder stores downloaded AI models such as:

- Stable Diffusion checkpoints
- AnimateDiff models
- Wav2Lip weights
- XTTS voice models
- MusicGen / AudioLDM models

## Purpose

Avoid re-downloading models every run.

## Structure example

models_cache/
├── stable_diffusion/
├── wav2lip/
├── xtts/
├── musicgen/
